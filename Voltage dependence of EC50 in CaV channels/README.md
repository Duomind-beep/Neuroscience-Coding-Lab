# EC50 vs Voltage Analysis (CaV Channels)

Exploratory computational analysis of EC50 dependence on membrane voltage in CaV channel data.

This notebook investigates how EC50 changes with voltage using a progression of models, from intuitive visualization to data-driven fitting.

Methods include:
- Raw data visualization of EC50 vs voltage
- Smooth vs switch-like behavioral comparisons
- Interactive parameter exploration (sliders)
- SciPy curve fitting (continuous model)
- Piecewise linear regression (regime analysis)
- Shared-breakpoint model at 0 mV

The final result highlights a clear difference in behavior between low- and high-voltage regimes, captured using both linear and unified piecewise models.

This project is exploratory and intended for learning computational modeling techniques in neuroscience data analysis.
